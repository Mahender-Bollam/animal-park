## Animal Traits 
This project demonstrates the implementation of various **Animal Traits** in Python, with the focus on **Cat** traits. It defines a general `Animal` class with various traits such as `eat`, `sleep`, and `sound`. Then, it showcases how these traits are applied to a specific animal, in this case, a **Cat**.

## Features

### Animal Class

The `Animal` class includes basic traits that can be inherited or modified for specific animals. These traits include:

- **Eat**: Defines the ability to eat.
- **Sleep**: Defines the ability to sleep.
- **Sound**: Defines the ability to make a sound (e.g., bark, meow).
- **Run**: Defines the ability to run.
- **Fly**: Defines the ability to fly.
- **Swim**: Defines the ability to swim.

Each of these methods will print the relevant action for an animal. The class serves as a base for other animals.

### Cat Class

The `Cat` class is a subclass of the `Animal` class and implements the following traits:

- **Eat**: "Cat can Eat"
- **Sleep**: "Cat can Sleep"
- **Sound**: "Cat can sound 'Meow'"

This class inherits from the `Animal` class and adds specific behaviors related to cats, like the sound they make.
