## Features
- Adds a new `Elephant` class to the animal park project.
- Contains specific attributes and behavior for elephants such as:
  - 4 legs
  - Gray color
  - Trunk
  - Trumpet sound

## Elephant Traits Example

```python
class Elephant:
    def __init__(self):
        self.legs = 4
        self.color = "gray"
        self.has_trunk = True
        self.sound = "trumpet"

    def traits(self):
        print("Elephants are amazing animals with trunks and trumpet sounds.")

```
